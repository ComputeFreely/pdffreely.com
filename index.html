Does PDF Freely upload PDFs?

No. PDF Freely reads and edits files in the browser and does not require an upload step.

Can PDF Freely merge and split PDFs?

Yes. PDF Freely can merge files, reorder pages, export selected pages, and download each page as a split ZIP.

Can PDF Freely fill or sign PDF forms?

No. PDF Freely is for page organization, page numbers, text stamps, watermarks, metadata, and image-to-PDF conversion. Use Fill Freely for PDF forms, text marks, checkmarks, dates, initials, and visual signatures.

Does PDF Freely include redaction?

No. Redaction is intentionally excluded because visual coverups can leave underlying PDF content extractable.
